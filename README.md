# Design It. — Product Customizer

A full-stack Django application that lets users upload a logo or design and instantly preview it on multiple product mockups (e.g. t-shirts, sweaters). The app uses **OpenCV** and **Pillow** for realistic image compositing — including background removal, displacement-map warping, and multiply-blend fabric textures.

---

## ✨ Features

| Feature | Description |
|---|---|
| **Catalog Generation** | Upload a design once and generate mockups for every product view in one click. |
| **Live Edit Preview** | Adjust position, scale, and rotation with real-time browser preview before saving. |
| **Colour Variants** | Switch between white and black product variants per mockup. |
| **Realistic Compositing** | Displacement-map warping + multiply-blend gives the design a printed-on-fabric look. |
| **Auto Background Removal** | White/near-white backgrounds are automatically stripped from uploaded designs. |
| **Batch Download** | Select multiple mockups and download them all as a single ZIP. |
| **Add Individual Products** | After initial generation, add extra product/angle/colour combinations on the fly. |
| **Admin Panel** | Full Django admin for managing products, product views, and print-area coordinates. |

---

## 🛠 Tech Stack

- **Backend:** Django 6.0, Django REST Framework
- **Image Processing:** OpenCV (headless), Pillow, NumPy
- **Frontend:** Vanilla HTML / CSS / JS (served via Django templates)
- **Database:** SQLite (dev) → PostgreSQL (production on Render)
- **Static Files:** WhiteNoise (compressed + hashed for caching)
- **Server:** Gunicorn
- **Hosting:** Render (Web Service + PostgreSQL + Persistent Disk)

## 🚀 Getting Started (Local)

### Prerequisites

- Python 3.11+
- pip

### Setup

```bash
# Clone the repository
git clone https://github.com/<your-username>/product_customizer.git
cd product_customizer

# Create and activate a virtual environment
python -m venv venv
venv\Scripts\activate        # Windows
# source venv/bin/activate   # macOS / Linux

# Install dependencies
pip install -r requirements.txt

# Run database migrations
python manage.py migrate

# Create a superuser (to access the admin panel)
python manage.py createsuperuser

# Start the development server
python manage.py runserver
```

Open **http://127.0.0.1:8000** in your browser.

### Adding Products

1. Go to **http://127.0.0.1:8000/admin/** and log in.
2. Create a **Product** (e.g. "T-Shirt").
3. Create one or more **Product Views** for that product:
   - Upload a base product image.
   - Set the **angle** (front / back / side) and **colour** (`#ffffff` or `#000000`).
   - Define the **print area** coordinates (`x`, `y`, `width`, `height`) — these specify where the user's design will be placed.

## 🌐 Deploy to Render

This project is pre-configured for **Render** with a `render.yaml` Blueprint.

### One-Click Deploy

1. Push the repo to GitHub.
2. In the [Render Dashboard](https://dashboard.render.com/), click **New → Blueprint** and connect your repo.
3. Render will read `render.yaml` and automatically create:
   - A **Web Service** (Python, Gunicorn)
   - A **PostgreSQL** database (free tier)
   - A **Persistent Disk** (1 GB) for uploaded media

### Environment Variables (auto-configured by Blueprint)

| Variable | Purpose |
|---|---|
| `SECRET_KEY` | Auto-generated by Render |
| `DEBUG` | Set to `false` |
| `ALLOWED_HOSTS` | `.onrender.com` |
| `DATABASE_URL` | Auto-injected from the managed PostgreSQL instance |
| `MEDIA_ROOT` | `/var/data/media` (persistent disk mount) |
| `PYTHON_VERSION` | `3.13.5` |

### Manual Deploy (if not using Blueprint)

```bash
# On Render → New Web Service
# Build Command:
./build.sh

# Start Command:
gunicorn core.wsgi:application
```

Set the environment variables listed above in the Render dashboard.

## 📡 API Endpoints

| Method | Endpoint | Description |
|---|---|---|
| `GET` | `/api/product-views/` | List all available product views |
| `POST` | `/api/generate-catalog/` | Generate mockups for all views of a colour variant |
| `POST` | `/api/customize/` | Generate a single mockup for a specific product view |
| `POST` | `/api/edit/<id>/` | Re-render a mockup with new transforms or colour |
| `GET` | `/api/download/<id>/` | Download a single result PNG |
| `POST` | `/api/download-batch/` | Download selected results as a ZIP |
